## What is TapCoupon?

TapCoupon lets businesses create branded coupon and loyalty cards. Cards can be physical (NFC) or digital (QR code via email). The API powers the redemption flow for your point-of-sale integration.

## Card types

| Type | How it's issued | How it's used |
| ---- | --------------- | ------------- |
| **Coupon card** | Physical (ordered in dashboard) or digital (issued at POS, sent by email) | Customer taps/scans → amount deducted from balance |
| **Loyalty card** | Physical or digital | Customer taps/scans → stamp recorded; reward triggered when target is reached |

## How it works

1. **Create a location** — represents a physical store or branch. Required before generating an API key.
2. **Create a template** — defines the rules for a card (e.g. "€50 gift voucher" or "5-stamp coffee loyalty card").
3. **Issue cards**
   - *Physical*: order NFC cards through the dashboard, then apply a template at the POS to activate them.
   - *Digital*: issue directly from the POS via the API — the customer receives an email with a QR code.
4. **Tap & redeem** — the POS calls the TapCoupon API to charge or stamp the card on each visit.
