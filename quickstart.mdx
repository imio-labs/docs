## Prerequisites

Before making API calls you need:

1. A TapCoupon account — sign up at the [Dashboard](https://dashboard.tapcoupon.at).
2. A **location** created in the dashboard (Settings → Locations).
3. An **API key** generated for that location (Settings → API Keys).

<Note>
  API keys are tied to a location. Each location you operate can have its own key.
</Note>

## 1. Log an employee in

Most card operations require an employee session. Log in with the employee's credentials to get a token.

```bash
curl -X POST "https://tapcoupon.at/api/v1/employee/login" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "username": "Employee Name",
    "password": "Employee Password"
  }'
```

```json
{
  "employee": { "id": "68d4f10eeceac90f5cf0ebd6", "username": "Employee Name" },
  "message": "Login successful",
  "token": "UmAQiB3ljRDyT51RuBIhCLscQMZVZTo3"
}
```

Save the `token` — you'll pass it as `X-Employee-Token` on subsequent requests.

## 2. Fetch a card

Retrieve a card's current state by its `cardId`.

```bash
curl "https://tapcoupon.at/api/v1/cards?cardId=9866191586296&timestamp=2026-01-01T00:00:00Z" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```json
{
  "cardId": "9866191586296",
  "cardType": "loyaltyCard",
  "activated": true,
  "scansNeeded": 5,
  "currentScans": 1,
  "rewardMessage": "You earned a free coffee!",
  "amountRenews": 0
}
```

## 3. Stamp a loyalty card

Record a scan when a customer completes a qualifying purchase. Amounts are in **cents**.

```bash
curl -X POST "https://tapcoupon.at/api/v1/loyalty-cards" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "X-Employee-Token: YOUR_EMPLOYEE_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{ "cardId": "9866191586296" }'
```

```json
{
  "success": true,
  "scansLeft": 4,
  "renewsLeft": 1,
  "message": "Scans completed: 1/5"
}
```

## 4. Charge a coupon card

Deduct an amount from a coupon card. All amounts are **integers in cents** (e.g. `500` = €5.00).

```bash
curl -X POST "https://tapcoupon.at/api/v1/coupon-cards" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "X-Employee-Token: YOUR_EMPLOYEE_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "cardId": "1885144192279",
    "amount": 500
  }'
```

```json
{
  "success": true,
  "amountLeft": 4500,
  "message": "Amount charged successfully"
}
```
