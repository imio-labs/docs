## GET `/customers`

Returns all customers whose cards have been issued or activated under your account.

### Authentication

`Authorization: Bearer YOUR_API_KEY`

### Example request

```bash
curl "https://tapcoupon.at/api/v1/customers" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

### Example response

```json
[
  {
    "name": "Bob",
    "email": "bob@example.com",
    "phoneNumber": "",
    "address": "Bakers Street 41",
    "zipCode": "1234",
    "city": "Vienna",
    "birthdate": "1990-06-15T00:00:00Z",
    "createdAt": "2026-02-01T15:59:34.647Z",
    "updatedAt": "2026-02-01T17:09:34.676Z"
  }
]
```

### Response fields

| Field | Type | Description |
| ----- | ---- | ----------- |
| `name` | string | Customer's full name |
| `email` | string | Email address |
| `phoneNumber` | string | Phone number (may be empty) |
| `address` | string | Street address |
| `zipCode` | string | Postal/ZIP code |
| `city` | string | City |
| `birthdate` | string | ISO 8601 date |
| `createdAt` | string | When the customer was first registered |
| `updatedAt` | string | Last time the customer record was updated |
