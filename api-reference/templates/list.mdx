## GET `/templates`

Returns all active templates associated with your API key's account. Templates define the rules for a card type and are required when applying or creating a card.

### Authentication

`Authorization: Bearer YOUR_API_KEY`

### Example request

```bash
curl "https://tapcoupon.at/api/v1/templates" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

### Template fields

| Field | Type | Description |
| ----- | ---- | ----------- |
| `id` | string | Use this as `templateId` when applying or creating cards |
| `templateType` | string | `"loyaltyTemplate"` or `"couponTemplate"` |
| `isActive` | boolean | Inactive templates cannot be applied to new cards |
| `cardData` | object | Default card values — amounts are in **cents** |
